# formation-page

1. Deploy Mkdocs 
After  ``` checkout ``` the primary working branch (master) of this git repository where i maintain the source documentation , run the following command:

```
mkdocs gh-deploy
```
Run Mkdocs on local http ( http://127.0.0.1 ) server with : 
```
mkdocs serve
```
